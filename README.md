# Reincarnated as a Dark Lord Before Taking Over the World with Power 999999999

An idle game with multiple endings and auto saves after finishing each run.

Control

    Click : loop action
    Right-click : single action
    Drag : pan camera
    Scroll : Zoom
    ESC : reincarnate

Is this game beatable without grinding?
Yes, by design you don't need Essence to reach all endings.
